Linkly EFTPOS

      

        

          

            

              
Terminal Connection

              
              Checking...
            

            

              Linkly Cloud works well with Wi-Fi pinpads: the terminal connects to Linkly over the network, and YieldPOS talks to Linkly Cloud using your paired credentials.
            

            

              

                
                
              

              

                
                
              

            

            

            

            

              
              
              
              
            

            

          


          

            
Wi-Fi Pinpad Checklist

            

              
1. Connect the pinpad to Wi-Fi or Ethernet and confirm it can reach Linkly Cloud. Terminal network setup should use Linkly Cloud on port 443.

              
2. Open the terminal's Linkly Cloud/POS pairing screen and generate a pairing code.

              
3. Enter the code here immediately. Linkly pairing codes expire after about 180 seconds, so refresh the code if pairing fails after a delay.

              
4. Pair once per physical terminal/register, then run a small live or sandbox test transaction.

            

          

        


        

          

            
Integration Notes

            

              YieldPOS uses Linkly Cloud endpoints for pairing, token creation, purchase, refund, settlement, and polling.
            

            

              
Sandbox auth:
auth.sandbox.cloud.pceftpos.com

              
Live auth:
auth.cloud.pceftpos.com

              
Sandbox API:
rest.pos.sandbox.cloud.pceftpos.com

              
Live API:
rest.pos.cloud.pceftpos.com
